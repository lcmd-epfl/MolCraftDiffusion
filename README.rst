MolecularDiffusion
==================

A molecular diffusion framework for machine learning applications.

Installation
-----------

Install from source:

.. code-block:: bash

    git clone https://github.com/pregHosh/MolecularDiffusion.git
    cd MolecularDiffusion
    pip install -e .

For development installation:

.. code-block:: bash

    pip install -e ".[dev]"

Usage
-----

Basic usage:

.. code-block:: python

    import MolecularDiffusion
    
    # Your code here

Command-line interface:

.. code-block:: bash

    moleculardiffusion --help

Documentation
------------

For more information, visit: https://moleculardiffusion.readthedocs.io
